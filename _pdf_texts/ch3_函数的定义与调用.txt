【例】输入三个整型数，用递推法求它们的最大公约数和最小公倍数。
对于两个整数的情况，假设有两个整数 num1 和 num2，最大公约数算法的步骤：
• 步骤1：计算 num1 除以 num2 的余数，记为 rem（即 rem = num1 % num2）。
• 步骤2：如果 rem = 0，则 num2 就是最大公约数。
• 步骤3：如果 rem ≠ 0，则令 num1 = num2，num2 = rem，并重复步骤1。
#include <iostream>
using namespace std;
int main() {
int num1 = 0, num2 = 0, LCM12 = 0; // 输入的两个数和最小公倍数
int num1Temp = 0, num2Temp = 0, temp = 0; // 计算用的中间变量
cout << "请输入两个整数: "; cin >> num1 >> num2;
num1Temp = num1; num2Temp = num2;
do {
temp = num1Temp % num2Temp;
num1Temp = num2Temp;
num2Temp = temp;
} while (temp != 0);
LCM12 = num1 * num2 / num1Temp;
cout << num1 << "和" << num2 << "的最大公约数是: " << num1Temp << endl;
cout << num1 << "和" << num2 << "的最小公倍数是: " << LCM12 << endl;
return 0;
