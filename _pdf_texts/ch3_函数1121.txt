前情回顾
无参函数定义格式为：
《数据类型》函数名(《void》){函数体}
数据类型：函数返回值类型，没有返回值应将返回值类型定义为void
函数名：采用合法标识符表示
对无参函数: 参数括号中的void可略，括号不能省略
函数体由一系列语句组成
有参函数定义格式为：
《数据类型》函数名 (参数类型1 形式参数1《,参数类型2 形
式参数2,…》){函数体}
形式参数简称形参，只能是变量名，不允许是常量或表达式；
各参数即使类型相同也必须分别加以说明。

【例】 设计函数，根据三角形的三边长求面积。如果不能构成三角形给出提示信息。
#include <iostream>
using namespace std;
充分利用函
float TriangleArea(float a, float b, float c) {
数的返回值
float s;
if ((a + b <= c) || (a + c <= b) || (b + c <= a))
return -1;
s = (a + b + c) / 2;
return sqrt(s * (s - a) * (s - b) * (s - c));
}
int main() {
float a, b, c, area;
cout << "输入三角形三边a,b,c:";
cin >> a >> b >> c;
area = TriangleArea(a, b, c);
if (area < 0)
cout <<'('<< a <<','<< b <<','<< c << ')' << "不能构成三角形！" << endl;
else
cout <<"三角形("<< a <<','<< b <<','<< c << ")面积为：" << area << endl;
}

函数的参数传递
引用（reference）
➢ C++的一种数据类型，也称为别名
➢ 不是定义新变量，而是给已经定义的变量取别名
定义：类型& 引用变量名 = 已定义过的变量名；
#include <iostream>
using namespace std;
int main() {
int a = 10;
int& b = a;
cout << "a = " << a << endl;
cout << "b = " << b << endl;
b = 100;
cout << "a = " << a << endl;
cout << "b = " << b << endl;
return 0;
}

#include <iostream>
#include <cmath>
using namespace std;
const double e = 1e-5;
const int Maxiter = 1000;
double Newton(double, double, int&);
int main() {
double a, x; int iter = 0;
cout << "输入一个正数: "; cin >> a;
x = Newton(a, a / 2.0, iter);
cout << a << " 的平方根为: " << x << "\n迭代次数为: " << iter << endl;
return 0;
}
double Newton(double a, double x, int& iter) {
double x0, x1;
x1 = x;
do {
x0 = x1;
x1 = (x0 + a / x0) / 2.0;
iter++;
} while ((fabs(x0 - x1) > e) && (iter < Maxiter));
return x1;
}
